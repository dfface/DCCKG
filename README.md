# DoctorKG

This is the code repository for DoctorKG.

Now, it concludes:

* Dataset Preprocess example
* BERT NER example

## Quick Start

```shell
conda create -n DoctorKG python=3.8
conda activate DoctorKG

pip install -r requirements.txt

python bert_ner.py

cat checkpoints/eval_results.csv
```
